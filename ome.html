OvenMediaEngine
Streaming Suite

Complete your pipeline with our optimized player and encoder.

OvenPlayer

A JavaScript-based player library optimized for WebRTC and LL-HLS streams from OvenMediaEngine, offering rich APIs for custom UX and tight integration with your service.

GitHub Guide

OvenLiveKit

A JavaScript-based live streaming encoder that wraps the WebRTC API into a rich, easy-to-use SDK for publishing sub-second latency streams to OvenMediaEngine with just a few lines of code.

Key Features

Sub-Second Latency

Delivers live streams in under 1 second using WebRTC, LL-HLS, and SRT.

(1/6)

Adaptive Bitrate (ABR)

Embedded live transcoder supports ABR for both WebRTC and LL-HLS.

(2/6)

High Scalability

Origin-Edge clustering architecture scales to accommodate massive traffic.

(3/6)

Various Ingests

Supports WebRTC, WHIP, SRT, RTMP, RTSP, MPEG-2 TS, RTSP Pull, and more.

(4/6)

REST API

Provides APIs for Access Control, Monitoring, and Integration.

(5/6)

Essential Features

DVR, DRM, Dump, Recording, Thumbnail, Restreaming, and more.

(6/6)
